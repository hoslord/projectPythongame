# projectPythongame

Проект игры Ping-Pong созданный на python.

В верхней части кода мы подключаем все библиотеки, задаем все глобальные переменные, инициализируем библиотеку pygame а также шрифты для добовления надписей в игру. Создаем платформы и шарик для игры.
Далее инициализируем игру, задаем параметры проигрыша и победы, настраиваем элементы игры(платформа, шарик). Создаем возможность переигровки и подсчета очков.
