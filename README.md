# projectPythongame

Описание проекта:
Проект игры Ping-Pong созданный на python.

Цель проекта:
Создание игры при помощи библиотеки pygame с полноценным функционалом и игровым процессом с элементом соревновательности.

Архитектура кода:
В верхней части кода мы подключаем все библиотеки, задаем все глобальные переменные, инициализируем библиотеку pygame а также шрифты для добавления надписей в игру. Создаем платформу и шарик для игры.
Далее инициализируем игру, задаем параметры проигрыша и победы, настраиваем элементы игры(платформа, шарик). Создаем возможность переигровки и подсчета очков. Завершаем игру последством команды.

Для запуска программы потребуется Docker. Можно установить с официального сайта. 
Создаем сборку командой docker build -t ping-pong . через Powershell
Запускаем сборку командой docker run --rm ping-pong
